About

Welcome to PitchFX.org, where we have a database of MLB atbat and pitch data going back to the 2010 season. This site provides a way for you to view that data for any pitcher or batter based on filters you define, such as pitch speed, pitch types, base-out states, and many others.

PitchFX.org is in it's infancy, but can be much, much more. Here is how you can participate:

Ideas

If you have an idea for a feature, file an issue on our GitHub page. We read every submission. If you find your idea has already been submitted, please comment and watch the issue. Issues that are generating a lot of interest will bubble to the top of the queue.

Code

PitchFX.org is open-source software. If you're a developer, I invite you to read our Developer Guide. Developers that contribute significantly to the project will get visibility in the Developers section.

The Data

Using the atbat-mongodb project, we pull pitch and at-bat data from each day's baseball games and store it in our own MongoDB database. Data is available back to the 2010 season.

Data from each day will be available by 6AM EST on the following day.

PitchFX.org is not affiliated with MLB, MLBAM or Sportsvision, although we are very grateful to those organizations for their technological advancements that have made sites like PitchFX.org possible.
